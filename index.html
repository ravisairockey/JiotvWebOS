Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://ravi-saivigneswara-2025.github.io/ (Give it the name "ravi sai vigneswara @ 2025 Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "ravi sai vigneswara @ 2025 Repository", and install repository.ravi-saivigneswara-2025.zip.
  • Restart Kodi
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "ravi sai vigneswara @ 2025 Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
