Install

  • Go to the Kodi file manager.
  • Click on "Add source".
  • Add: https://ravi-saivigneswara-2025.github.io/ as "ravi sai vigneswara @ 2025 Repo".
  • Go to "Addons" → "Install from zip" → select "ravi sai vigneswara @ 2025 Repository" → install repository.ravi-saivigneswara-2025.zip.
  • Restart Kodi.
  • Now, "Install from repository" → "ravi sai vigneswara @ 2025 Repository".
  • Go to "Video add-ons" and install your desired add-ons.
